Static Image Display

A simple website to showcase an image. Easily customizable by updating the image source in the code.

This is a sample image displayed on the website. To change this image, simply update the src attribute in the HTML code.

How to Change the Image

To display your own image:

  1. Locate the <img> tag in the HTML code (around line 80)
  2. Replace the value of the src attribute with the path to your image
  3. Update the alt attribute with a description of your image
  4. Save the file and refresh your browser

Example: If your image is named "my-photo.jpg" and is in the same folder as this HTML file, change the src to src="my-photo.jpg".
